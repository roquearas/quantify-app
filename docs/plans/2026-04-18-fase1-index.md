# Fase 1 — Índice de Sub-Planos

**Spec**: [`docs/specs/2026-04-18-fase1-fundacao-design.md`](../specs/2026-04-18-fase1-fundacao-design.md)

A Fase 1 é grande (~3-4 semanas). Foi decomposta em 6 sub-planos sequenciais. Cada um produz software testável e commitável isoladamente.

| # | Sub-plano | Cobre do spec | Status | Estimativa |
|---|---|---|---|---|
| **1A** | [Reconciliação de schema + trigger service_request → project](2026-04-18-fase1a-reconciliacao-schema.md) | §4.1, §4.2 | ✅ Completo (2026-04-18) — ver [log do teste](fase1a-teste-trigger.md) | 2-3 dias (feito em 1 sessão) |
| **1B** | [HITL workflow + UI de review item-por-item](2026-04-18-fase1b-hitl-workflow.md) | §4.3 | ✅ Completo (2026-04-18) — ver [log do teste](fase1b-teste-hitl.md) | 3-4 dias (feito em 1 sessão) |
| **1C** | [Gerador de PDF do orçamento](2026-04-18-fase1c-pdf-generator.md) | §4.4 | ✅ Completo (2026-04-18) | 3-4 dias (feito em 1 sessão) |
| **1D** | Organização de documentos por project | §4.5 | ⏳ Pendente | 1-2 dias |
| **1E** | Dashboards com dados reais | §4.6 | ⏳ Pendente | 2 dias |
| **1F** | Templates para 3 serviços + E2E + polimento | §4.7, §4.8 | ⏳ Pendente | 4-5 dias |

**Ordem**: deve ser seguida sequencialmente — 1A destrava 1B, que destrava 1C e 1E em paralelo, que destravam 1D, que destrava 1F.

**Escrita dos planos**: o sub-plano N+1 é escrito quando o N é completado, para que mudanças descobertas durante a execução sejam incorporadas.

## Critérios globais de "Fase 1 concluída"

Repete os critérios do spec (§11):

- [ ] Schema reconciliado (todas as migrations aplicadas e batendo com código)
- [ ] Fluxo completo em staging: solicita → valida → PDF → paga → baixa PDF final
- [ ] 3 tipos de serviço (obra, elétrica, laudo) com templates próprios
- [ ] Trilha em `validations` para todo budget VALIDATED
- [ ] Dashboards com dados reais (sem mock)
- [ ] 3 cenários E2E passando no CI
- [ ] Teste RLS cross-tenant falha como esperado
- [ ] 1 orçamento real validado por engenheiro humano
- [ ] Docs atualizados (README, setup-guide, spec com "Implementado")
